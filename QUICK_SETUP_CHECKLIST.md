### Step 2: Create Firestore Database (1 minute)

1. Open: https://console.firebase.google.com/project/bilu-store-e1a06/firestore
2. Click "Create database" button
3. Select "Production mode" → Click "Next"
4. Location: Select "europe-west1 (Belgium)" → Click "Enable"
5. Wait ~1 minute for provisioning (you'll see a loading spinner)
